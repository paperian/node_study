for (var i ~ ) 로 인해 i가 전역 변수가 될 수...

따라서 이크마6에서는

변수를 생성할 땐 let

상수를 생성할 땐 const

쓰는 원칙은

일단 const를 쓰고 오류가나면 let

이유는 오브젝트 같은 경우엔 주소값이기 때문에

const로 주소값만 알아도 멤버 변수를 수정할 수 있음.

숫자 문자열 같은 경우엔 변경할 필요가 없다면 const로 선언

var 과 같은 경우 한 스코프 내에서 여러번 선언 가능

let 으로 선언하면 에러남 (already been declared)

	var a = 10;
}
console.log(i)
console.log(a)
