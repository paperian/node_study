express 모듈 : http 모듈처럼 사용할 수 있고, 훨신 더 많은 기능 포함한 외부 모듈
; npm install express@4

//////////////////////////////////////////////////////////////////////////////////////////////////
미들웨어 : express 모듈 use() 메서드에 입력하는 함수들!
//////////////////////////////////////////////////////////////////////////////////////////////////
router : 페이지 라우팅 지원 미들웨어
static : 지정한 폴더에 있는 내용을 모두 웹 서버 루트 폴더에 올릴 때 사용
morgan : 웹 요청이 들어왔을 때 로그 출력
cookie parser : 요청 쿠키를 출력
body parser : POST 요청 데이터를 추출
connect-multiparty : multipart/form-data 인코딩 방식을 사용해 POST 요청 데이터를 추출
express-session : 세션을 쉽게 생성
//////////////////////////////////////////////////////////////////////////////////////////////////
