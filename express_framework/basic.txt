express '프로젝트명'

했을 때, 생기는 폴더 및 파일의 역할

bin 폴더 ; 프로그램의 실행과 관련된 파일이 들어있는 폴더다. 
					 이 폴더 내부의 www파일을 실행해서 프레임워크 실행

public 폴더 ; express 모듈의 static 미들웨어를 사용해 웹 서버에 올라가는 폴더이다. 
						  이 폴더에 자바스크립트, css, 그림 등의 리소스 파일을 생성한다.

routes 폴더 ; 페이지 라우트와 관련된 모듈, index.js 파일과 routes 파일이 있음

views 폴더 ; ejs 파일 또는 jade 파일과 같은 템플릿 파일 저장

app.js 파일 ; 프로젝트에서 중심이 되는 파일

package.json ; 현재 프로젝트와 관련된 정보 및 모듈 정보
